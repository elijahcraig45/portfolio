## 🌟 Flagship Project: Hank's Tank

### What It Is
A production-grade MLB analytics platform with real-time dashboards, REST API, and daily ML predictions.

### Components

#### 🎥 Frontend (React)
- Live game strike zones with D3
- Player statistics and comparisons
- Team performance dashboards
- Real-time data updates
- **Tech**: React 18, Bootstrap 5, D3.js, Recharts

#### 🔌 Backend API (TypeScript/Express)
- REST API serving 35,000+ records
- BigQuery data integration
- ML prediction serving
- Request deduplication & caching
- **Tech**: TypeScript, Express, BigQuery, Cloud Functions

#### 🤖 ML Pipeline (Python)
- Feature engineering (113 features)
- Ensemble models (XGBoost, LightGBM, CatBoost)
- Daily inference pipeline
- Validation & testing
- **Tech**: Python, BigQuery, Cloud Functions, Cloud Scheduler

### Key Metrics
- **Model Accuracy**: 57.65% on 2025 holdout
- **Historical Data**: 35,000+ records
- **Engineered Features**: 113 advanced features
- **Services**: 3 interconnected systems
- **Uptime**: Production-grade deployment

## 🚀 Getting Started

### View Live
Just visit: https://elijahcraig45.github.io/portfolio/

### Local Development
```bash
# Clone repository
git clone https://github.com/elijahcraig45/portfolio.git
cd portfolio

# Start local server (choose one)
# Python 3
python -m http.server 8000

# Node.js
npx http-server

# Open browser
# http://localhost:8000
```

### Make Changes
1. Edit `index.html` for content
2. Edit `assets/css/styles.css` for styling
3. Test locally before pushing
4. Commit and push to GitHub
5. Changes deploy automatically in 1-2 minutes
